# Watch Apps

Ce dossier contient les applications Garmin embarquees de `nouvelleApp`.

Structure cible:

- `fenix-connect-iq`
- `edge-connect-iq`

Principe:

- chaque appareil Garmin garde son app `Connect IQ` dediee
- le contrat de donnees reste aligne avec `OpenSpec/11-change-watch-poc-data-contract.md`
- le companion mobile vit dans `apps/mobile`
