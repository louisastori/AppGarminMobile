# Garmin Connect IQ Companion

Ce module pose le squelette du companion mobile pour les apps `Connect IQ` Garmin.

Responsabilites cibles:

- handshake appareil
- reception des lots montre / Edge
- acquittement des batches
- persistence locale
- exposition d'un statut de lien lisible par l'UI
